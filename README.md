## About me
My academical background is M.Sc. in Data-Centric Engineering, majoring in Computer Vision and Pattern Recognition. In my Master's studies I focused on the following consepts:
- Advanced data analysis, data preprocessing, machine learning and pattern recognition
- Image preprocessing, digital imaging, image analysis, computer vision
- Bayesian statistics, continuous and statistical parameter estimation numerical methods for PDEs, 

Currently I'm working full-time as a full-stack software developer, mostly dealing with .NET, C#, Visual Basic, Javascript and SQL Server. AWS and Docker are also somewhat familiar to me due to my work.

I'm enthusiastic about data science related projects, I love to play with data and find meaningful insights in it. I have always a few hobby projects in development, and even more ideas about new projects on my mind. Some highlighted projects I have had are listed below, feel free to take a look. Some repositories have a better documentation than others...
